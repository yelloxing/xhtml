## Installation

In a browser:
```html
<script src="./dist/xhtml.js"></script>
```

use:
```js
__.XXX();
```

Using npm:
```shell
$ npm i xhtml.js
```

Note: add --save if you are using npm < 5.0.0

```js
const __ = require('xhtml.js');
```

or

```js
import __ from 'xhtml.js';
```
