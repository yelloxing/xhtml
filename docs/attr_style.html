属性
获取或设置结点属性： 
xhtmlObj.attr(attr[, val]);
如果val没有传，表示获取属性值，否则为设置属性值。
样式
获取或设置样式： 
xhtmlObj.css();
样式的设置参数比较复杂，一共分以下四种情况：
1.arguments(key):获取指定样式
2.arguments(key,value):设置指定样式
3.arguments():获取全部样式
4.arguments(json):设置大量样式
温馨提示：获取的样式是浏览器渲染后的值，比如你设置color：red，使用此方法返回的就是rgb(255, 0, 0)！
class
1.判断是否有： 
xhtmlObj.hasClass(classString);
2.添加： 
xhtmlObj.addClass(classString);
3.删除： 
xhtmlObj.removeClass(classString);
4.切换（有删除，没有添加）： 
xhtmlObj.toggerClass(classString);
