追加结点
1.在被选元素内部的结尾插入内容： 
xhtmlObj.append(node|template);
2.在被选元素内部的开头插入内容： 
xhtmlObj.prepend(node|template);
3.在被选元素之后插入内容： 
xhtmlObj.after(node|template);
4.在被选元素之前插入内容： 
xhtmlObj.before(node|template);
查找结点
1.后代结点： 
xhtmlObj.find(tagName, checkback);
2.祖先结点： 
xhtmlObj.parents(checkback, stopback);
3.孩子结点： 
xhtmlObj.children(checkback);
4.指定序号结点： 
xhtmlObj.eq(index);
删除结点
1.删除当前结点： 
xhtmlObj.remove();
